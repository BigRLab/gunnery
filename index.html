Gunnery

Remote execution frontend

Gunnery is multipurpose task execution tool for
distributed systems with an easy to use interface

Distributed

Execute custom commands on distributed, multistaging environments

Multipurpose

Usable for application deployment, service control or backups

Flexible

Supports capistrano, ant, phing, fabric, puppet, and that other tool you use too

Secure

Responsive interface

Open Source

 

How it looks like?

How it works?

Gunnery is build on top of Django framework and uses Celery for background tasks. Other important requirements are PostgreSql database and RabbitMQ queue system.

How to install?

Step by step instructions are available on wiki page. Included Puppet manifests allow fast way to start Gunnery instance (full-stack).
